About this platform
E-voting system is designed to enable Electoral Authority of the teachers to use capture (teachers) voter’s information. The design cover security issues like authenticating the ID number and Password of the voters (teachers) through integrating the ID and password database to the voters’ registration process and also capturing the picture of the electorates for validation during election. The project covers all common electoral supports/functionality to voter and authority.
This surely increases the voting percentage in university. E-Voting System will be fast enough to calculate the results and reduce the human efforts, as all the things will be automated.
